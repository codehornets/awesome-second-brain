# Templates

Use these templates when adding or refreshing solution profiles and capability definition pages.

| Template | Use for |
|---|---|
| [system-profile.md](system-profile.md) | A solution profile under `solutions/`. |
| [capability-page.md](capability-page.md) | A capability definition under `capabilities/`. |

## Workflow

1. Copy the relevant template into `solutions/` or `capabilities/`.
2. Replace placeholders with source-backed claims.
3. Leave unsupported or unverified claims as `Unknown`.
4. Link the new page from README, the folder README, and comparison pages.
