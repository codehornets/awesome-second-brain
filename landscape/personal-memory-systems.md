# Personal / Local Memory Systems

Personal and local memory systems focus on persistent knowledge about an individual, their projects, their documents, and their history.

## Systems

- [Gbrain](../systems/gbrain.md)
- Basic Memory
- Claude-mem
- LLM Wiki
- Reor
- Obsidian-based memory systems
- OpenMemory

## Research Questions

- Is this primarily a human-readable second brain or an agent-facing memory layer?
- How does it ingest notes, files, messages, or user events?
- Does it maintain a knowledge graph, semantic index, timeline, or raw archive?
- Can agents write to it directly?
- Is it local-first, cloud-hosted, or self-hosted?
- Can memory be shared with a team or across multiple agents?

## Landscape Notes

Personal memory systems are often strongest when the goal is building a durable personal brain. The key comparison point is whether the system is optimized for human knowledge management, agent operations, or both.
