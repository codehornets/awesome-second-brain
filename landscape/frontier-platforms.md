# Frontier AI Platforms

Frontier AI platforms provide memory and personalization inside major AI applications. These systems are often easy for end users to adopt, but the memory usually stays inside the platform.

## Systems

- [ChatGPT Memory](../systems/chatgpt-memory.md)
- Claude Projects and personalization
- Gemini Saved Info
- Perplexity Spaces / personalization
- Grok personalization

## Research Questions

- What does the platform remember about the user?
- Is memory automatic, manually configured, or both?
- Can users view, edit, delete, or export memories?
- Does memory work across the platform's web, mobile, API, or agent surfaces?
- Can other agents or tools access the memory?
- How transparent is the retrieval behavior?

## Landscape Notes

Frontier platform memory is important because it makes personalization mainstream. The tradeoff is that memory is usually platform-bound and difficult to reuse across other agents.
