# Memory Infrastructure & SDKs

Memory infrastructure products provide APIs, SDKs, and frameworks that developers use to add long-term memory to applications or agents.

## Systems

- Mem0
- Zep
- Letta
- LangGraph memory
- LlamaIndex memory
- Custom vector database or knowledge graph memory stacks

## Research Questions

- What memory primitives does the system expose?
- Does it support user, session, team, or organization scopes?
- How are memories extracted, ranked, updated, expired, or deleted?
- What retrieval APIs are available?
- Does it provide observability and user control?
- How hard is it to integrate into an existing product or agent runtime?

## Landscape Notes

Infrastructure systems matter because they shape how new AI products add memory. The main question is whether they solve only application memory, or also portability across tools and agents.
