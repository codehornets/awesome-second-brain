# Watchlist

아래 프로젝트들은 AI 세컨드 브레인 아키텍처와 관련이 있지만, 아직 이 레포에서 전체 프로필로 평가하지 않은 항목입니다. 출처 기반 조사와 명확한 공식 설정/평가 링크가 준비된 뒤에만 `solutions/`로 옮기세요.

| 프로젝트 | 중요한 이유 | 현재 상태 |
|---|---|---|
| Letta / MemGPT | memory block, archival memory, agent-managed context를 가진 stateful agent architecture. | agent-framework memory 기준선 후보. |
| Mem.ai | API/MCP와 consumer knowledge assistant positioning을 가진 AI notes app. | AI notes / PKM product category 후보. |
| Tana | Supertag와 AI command를 가진 structured knowledge graph notes. | schema-heavy PKM category 후보. |
| Reflect | backlink와 graph-oriented notes를 가진 encrypted AI notes app. | AI notes / PKM category 후보. |
| Capacities | object-based knowledge studio. | object-centric PKM category 후보. |
| TideMind | knowledge graph claim을 가진 local-first AI second-brain positioning. | emerging local-first 후보. |
| Dory | MCP-aware agent를 위한 local-first shared memory daemon. | emerging local-first 세컨드 브레인 후보. |
| Open Second Brain | agent를 위한 Obsidian-native memory layer. | emerging Obsidian/MCP 후보이며 OpenHuman과 별개. |
| MAGI | AI agent를 위한 persistent memory. | emerging AI-memory 후보. |

## 승격 규칙

watchlist 항목이 핵심 프로필이 되려면 아래가 준비되어야 합니다.

- 공식 문서 또는 저장소 근거
- 배포 모델
- 설정 부담과 공식 설정 링크
- AI 도구 활용 표면
- 맥락 수집과 검색/활용 모델
- 알려진 한계
- 트레이드오프
