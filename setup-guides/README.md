# Setup Guides

Setup guides are hands-on, repo-owned evaluation notes for getting a second-brain workflow running. They should not replace official product docs.

Add a setup guide only when it includes:

- the goal and target user
- the exact solution and client tested
- prerequisites and accounts needed
- the shortest verified path to first useful memory
- how to verify that retrieval or write-back actually works
- known failure modes or setup tradeoffs

Keep official installation details as links unless this repo verified a specific setup path.
